# Circular Progress Bar

A circular progress bar built with pure HTML & CSS using `conic-gradient`.

## Demo

[Live demo on GitHub Pages](https://yura-posledov.github.io/progress/)

## Screenshot

The component displays a coral-colored ring progress indicator (~68%) on a dark card, with a Mars landscape background.

## Tech

- HTML5
- CSS3 (`conic-gradient`, flexbox)
- No JavaScript
